# Developer Base

Испольую данный репозиторий для создания минимальной документации к различным инструментам, такаим как `k8n`, `kafka`, `rabbit` и прочим.

Некоторые модули базы знаний, например [rabbit](./rabbitmq/example) и [CI/CD](./cicd/example) оснащены моими примерами.

Содержание:
- [CI/CD](./cicd/) - непрерывная интеграция и доставка
- [GraphQL](./graphql/) - язык запросов для API
- [Kafka](./kafka/) - платформа для обработки потоков данных)
- [RabbitMQ](./rabbitmq/) - платформа для обмена сообщениями
- [SQL](./sql/) - язык запросов для базы данных
- [UV](./uv/) - утилита для работы с виртуальными окружениями
- [Linux](./linux/) - базовые команды для работы с Linux
